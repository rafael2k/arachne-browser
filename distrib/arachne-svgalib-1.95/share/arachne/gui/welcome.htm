Welcome to Arachne ! version:
Arachne is fullscreen graphical WWW browser for Linux and DOS, which means that:
  • it can view hypertext documents with inlined images (HTML/4.0 language)
  • it can browse World Wide Web (HTTP/1.0 protocol over TCP/IP connections)
  • it doesn't require any Windows, OS/2 or X11 to run on your PC
    But that's not the end ! Arachne can also:
  • send and recieve Internet e-mails (MIME compatible)
  • edit source code of local HTML pages
  • perform FTP downloads and uploads
  • call many DOS applications as plug-ins
    • Thank you for installing Arachne. We hope that you will find Arachne useful and that you will decide to use it as your Internet or Intranet client.
