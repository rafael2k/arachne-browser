How to register Arachne

  • Arachne is distributed as a FREE software package that contains a SHAREWARE evaluation version of the browser itself and various FREEWARE or PUBLIC DOMAIN utilities. Nobody should charge more for redistribution of the package than the cost of the media and copying expenses.
  • Without registering, you can use the copyrighted parts of Arachne only for personal or NON-COMMERCIAL purposes. If you want to use Arachne for other purposes, for example distribute it as a part of your product, you MUST register. Some of the features of the registered version are not available without the registration key. Currently these features are:
    • Full customization of the eMail signature
    • Online time counter (files online.log and ppptotal.log)
    • The ability to disable annoying banners (images of size 468x60 pixels)
    • Various OEM customizations
    • Toolkit for customization of the Arachne user interface.
  • Unless you register, we will not provide you any technical support, but we try to analyse and solve problems REGISTERED users experience with Arachne.
  • You can register the Arachne WWW browser by buying a REGISTRATION KEY from xChaos software or from any authorised distributor. Licensing for OEM devices must be discussed individually with the author (contact sales@arachne.cz)
