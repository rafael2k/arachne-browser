Input fields and text windows are edited using built-in Arachne editor. In editor, you can use standard editing keys (cursor keys, <Backspace>, <Delete>, <Home>, <End>, <Page Up>, <Page Down>). <Shift> + cursor keys marks block, and following commands are available:
<Ctrl+X> Cut to clipboard (*) Left mouse button Move/mark block (*)
<Ctrl+V> Paste from clipboard (*) Middle mouse button Paste block
<Ctrl+C> Copy to clipboard (*) Right mouse button Copy to clipboard
<Ctrl+Page Up> Goto top of file (*) <Ctrl+K> B Begin block
<Ctrl+Page Down> Goto end of file (*) <Ctrl+K> K End block
<Ctrl+Home> Page Left <Ctrl+K> M (V) Move block
<Ctrl+End> Page Right <Ctrl+K> Y Delete block
<Ctrl+Left> Previous word <Ctrl+K> C Copy block
<Ctrl+Right> Next word <Ctrl+K> H Hide block
<Ctrl+T> Toggle wordwrap (*) <F7> (<Ctrl+Q>) Search for string
<Insert> Toggle insert/overwrite <Ctrl+L> Search next
<F2> Submit form or save file (*) <Ctrl+Y> Delete line
<F3> (<Ctrl+R>) Insert file (block) (*) Left doubleclick Mark line (*)
<Ctrl+W> Write to file (*) Unmark block (*)
<Tab> Goto next input field Submit URL
<Shift+Tab> Goto previous input field Right doubleclick Delete block (*)
<Enter> Goto next input field (**)
<Esc> Exit editor mode
(*) Multi-line windows only (**) Submits form if in last single-line input field in form
