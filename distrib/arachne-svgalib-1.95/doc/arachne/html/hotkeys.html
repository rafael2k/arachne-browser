(*) You can add selected URL to hotlist by pressing A, paste it to text field or (optionaly) use Delete key.
(**) In multiple line text input fields, left double-click marks line as block and instantly copies it to clipboard. If block is marked, left doubleclick unmarks it. Left button single click moves cursor, left button drag marks block. In URL input field, left double click works like pressing Enter key.
(***) Right double-click cuts marked block like Ctrl+X, while right single click positions cursor and copies previously marked block to clipboard like Ctrl+C In single line fields, right click always clears entire field like Ctrl+Y
