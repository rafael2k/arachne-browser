Embeded sound file

This is example of sound file embeded in HTML page. Any audio format defined in MIME.CFG can be embeded using <EMBED> tag. While the sound is being played, user can't do anything else. The situation is better for memory resident MIDI players, as calling the player will start the playback on background, and then Arachne will restart. In this case, there is problem how to stop the music: solution can be defining program for unloading MIDI player from memory as DGI script. Only recommended for experienced Arachne and DOS users.

Note 1: it's obvious, that this example will work only if you have correct plugin for playing WAV files on your audio device...

Note 2: this feature is currently available only for local web pages (file: protocol).
