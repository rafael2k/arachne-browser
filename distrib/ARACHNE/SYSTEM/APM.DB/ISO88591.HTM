ISO-8859-1 fonts

 License: Public domain software
This package contains ISO-8859-1 (Latin-1) fonts. Created for West European Arachne users by Krzysztof Zietara from Poland. This is an early version: please report any "weird looking" characters to author.

ISO-Latin-1 entities declared in fontinfo.bin file and size 1 fonts were added by me Michael Polak. Entities conform RFC1866 standard.

If you have suitable keyboard driver, you can set outgoing charset to ISO-8859-1 (+text encoding either 8bit or quoted-printable), so you can use accented characters in eMail messages.
