Arachne setup tools

Arachne offers various setup tools and wizards that modify arachne.pck and arachne.cfg for you. This is a list of what actually happens during various phases of the setup process.

Standard installation process

  • The self-extracting distribution archive launches setup.bat
  • If setup.bat doesn't find any arachne.cfg file in the current directory,
  • setup.bat launches CORE.EXE in basic memory and video setup mode
  • setup.bat launches arachne.bat and calls setup_os.dgi, which detects the computer speed and suggests a new initial profile and changes to AUTOEXEC.BAT/CONFIG.SYS
  • The user is pointed to the TCP/IP setup screen.

    Subsequent re-running of setup.bat

  • setup.bat finds arachne.cfg file
  • setup.bat launches CORE.EXE in basic memory and video setup mode
  • The user is pointed to the TCP/IP setup screen.
  • If necessary, setup_os.dgi can be launched manually from Desktop | Your Applications.

    Note: this also happens if you delete arachne.pck and then run arachne.bat.

    Launching Desktop | Your Applications | setup_gr.ook

  • setup_gr.ook launches CORE.EXE in basic video setup mode and
  • Arachne restarts and returns to the previous page

    Launching Desktop | Your Applications | setup_os.dgi

  • setup_os.dgi detects the computer speed and suggests a new initial profile and changes to AUTOEXEC.BAT/CONFIG.SYS
  • The user is pointed to the TCP/IP setup screen.

    Note: arachne.cfg is overwritten in this case - use ACF profiles.

    Hitting the 'S' key

  • The user is pointed to the TCP/IP setup screen.

    Hitting the 'O' key or going to Desktop | Options

  • The user is pointed to the Arachne Options screen.
  • The TCP/IP setup screen can also be accessed from this screen.
