Macintosh file, packed by BINHEX

To unpack this file, use the HEX2BIN.EXE utility from the command line (you have to install BINHEX.APM). with the name of the packed file as argument. To save the converted file, press F2 and select window "Save as..."
If you wish to forward this file to a Macintosh user, send the packed file without modifications. To send it right then, press F2 and select window "Send document..."
