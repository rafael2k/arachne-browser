Arachne and DOSSHELL

DOSSHELL is Microsoft's official GUI for MS-DOS versions cca 6.x and older. Arachne uses enviroment variables extensively, and DOSSHELL had problems with that. This is solution of the problem:

You can boot ARACHNE within DOSSHELL, if the autoexec.bat file
contains

SET ARACHNE=vvvvvvvvvvvvvvvv
SET ARACHNETEMP=vvvvvvvvvvvvvvvvvvvvvvvv
SET MYIP=vvvvvvvvvvvvvvvvvvvvvvvvvvv
SET REMIP=vvvvvvvvvvvvvvvvvvvvvvvvvvvvvv
SET NETMASK=vvvvvvvvvvvvvvvvvvvvvvvvvvv
SET PEERMRU=vvvvvvvvvvvvvvvvvvvvvvvvvv

This allocates environment space before entering DOSSHELL. Then all
is fine. (But, do not use task swapper!)
